hexo和github搭建个人博客

1.安装git

https://note.youdao.com/s/1sXuK0lP

2.安装node.js环境‘

https://note.youdao.com/s/ZVRPmaav

3.链接github

右键打开git-bash
然后输入命令：

git config --global user.name "GitHub 用户名"
git config --global user.email "GitHub 邮箱"

我的是这样的：

git config --global user.name "Romantic-1"
git config --global user.email "2751587017@qq.com"

然后创建ssh密钥：
输入 ssh-keygen -t rsa -C “GitHub 邮箱”，然后一路回车。

ssh-keygen -t rsa -C "2751587017@qq.com"

此时在电脑本地就会生成密钥和公钥，将公钥打开复制

然后到个人GitHub网站保存这个公钥


验证链接
打开git-bash，输入命令：ssh -T git@github.com
会提示输入yes，回车，出现successful则表示链接成功

如果出现连接失败，输入命令查看链接过程：
ssh -vT git@github.com
如果出现的是127.0.0.1失败
表示dns解析出现了问题
解决办法：
参考链接：https://juejin.cn/post/7103738143513247781
我使用的是steam++加速器，在加速之前，编辑hosts文件，添加以下内容到末尾

20.205.243.166  github.com
20.205.243.160  ssh.github.com

3a.创建 Github Pages 仓库

GitHub 主页右上角加号 -> New repository：

Repository name 中输入 用户名.github.io
勾选 “Initialize this repository with a README”
Description 选填
填好后点击 Create repository 创建。

5.本地环境安装hexo

新建一个文件夹用来存放 Hexo 的程序文件，如 Hexo-Blog。打开该文件夹，右键 -> Git Bash Here。
使用 npm 一键安装 Hexo 博客程序：
npm install -g hexo-cli
这个命令也可以：
npm install -g hexo #开始安装hexo
创建存放博客的目录#初始化创建 test 文件夹
hexo init test
进入test目录
cd test
然后安装hexo所需的文件
npm install
然后生成静态网页
hexo generate
然后启动本地服务
hexo server 或者hexo server -p 80

此时可以查看localhost:4000到网页服务

本地博客测试成功后，就是上传到 GitHub 进行部署，使其能够在网络上访问。

首先安装 hexo-deployer-git：
在博客的目录中打开git-bash安装

npm install hexo-deployer-git --save

完成后运行 hexo d 将网站上传部署到 GitHub Pages。
hexo deploy

完成！这时访问我们的 GitHub 域名 https://用户名.github.io 就可以看到 Hexo 网站了。

7.更换hexo主题

下载主题压缩包
https://hexo.io/themes
https://blog.lixiaomu.fun/posts/43857/
把下载的主题解压，放到 C:\Users\你的用户名\test\themes 文件夹里面（博客目录文件夹里面），然后用编辑器 (如:Notepad++) 打开 \test_config.yml 文件。在里面找到 theme 修改成你的主题名称，必须和主题文件夹的名称一致。

然后需要清除缓存和重新生成

hexo clean #清除缓存
hexo generate #生成静态网页

然后再重新推送到github
hexo deploy

此时可能会出现一个问题，更换主题后页面会显示下面这个提示

extends includes/layout.pug block content     #recent-posts.recent-posts include includes/recent-posts.pug include includes/pagination.pug

解决办法：
https://zhuanlan.zhihu.com/p/137946156
1、在hexo目录下打开git bash，输入命令：

npm install --save hexo-renderer-jade hexo-generator-feed hexo-generator-sitemap hexo-browsersync hexo-generator-archive

2、然后清除缓存，重新生成，推送

hexo clean
hexo generate
hexo deploy

8.发布新文章

进入博客所在目录，右键打开 Git Bash Here，创建博文：

hexo new "My New Post"

然后 source 文件夹中会出现一个 My New Post.md 文件，就可以使用 Markdown 编辑器在该文件中撰写文章了。

写完后运行下面代码将文章渲染并部署到 GitHub Pages 上完成发布。以后每次发布文章都是这两条命令。

hexo generate   # 生成页面
hexo deploy   # 部署发布

9.博客网站设置

包括网站名称、描述、作者、链接样式等，全部在网站目录下的 _config.yml 文件中，参考官方文档按需要编辑。
注意：冒号后要加一个空格！

设置网站图标
进入 themes/主题 文件夹，打开 _config.yml 配置文件，找到 favicon 修改，一般格式为：favicon: 图标地址。（不同主题可能略有差别）

修改并部署后没有效果
使用 hexo clean 清理后重新部署。
